# Daily Routine Examples

These examples use the V2 resource set. They are workflow patterns, not instructions to abuse systems.

## 1. Morning Research Sprint

1. Start with **Perplexity**, **Semantic Scholar**, or **Google Scholar** for broad discovery.
2. Use **Unpaywall**, **CORE**, **Open Access Button**, **DOAJ**, **arXiv**, or **PMC** for legal full-text access.
3. Push candidate papers into **Elicit**, **Consensus**, or **Scite** for extraction and validation.
4. Save high-value sources into **NotebookLM**, **Gistr.so**, **Obsidian**, or **Notion**.
5. Archive volatile pages with **Wayback Machine** or cautiously with **Archive.today**.

## 2. Defensive OSINT Check

1. Map domains with **BuiltWith**, **SecurityTrails**, **crt.sh**, and **ARIN Whois**.
2. Check public exposure with **Shodan**, **Censys**, **FOFA**, **ZoomEye**, **LeakIX**, or **FullHunt**.
3. Check breach exposure with **Have I Been Pwned** and, where authorized, **DeHashed** or **Intelligence X**.
4. Inspect suspicious links with **urlscan.io**, **VirusTotal**, **Hybrid Analysis**, and **Lookyloo**.
5. Document findings in **tldraw** or **diagrams.net**.

## 3. Design / Content Batch

1. Pull legal media from **Unsplash**, **Pexels**, **Pixabay**, **Freepik**, **Vecteezy**, or **Vectorportal**.
2. Clean/edit in **Photopea**, **Cleanup.Pictures**, **Remove.bg**, or **TinyWow**.
3. Build palette and structure with **ColorSpace**, **tldraw**, and **diagrams.net**.
4. Package posts with **AICarousels**, **CapCut**, **Descript**, **Splice**, or **InShot**.
5. Use **Listnr AI** or **ElevenLabs** only with disclosure and consent when voices matter.

## 4. Builder / Automation Block

1. Sketch the flow in **tldraw** or **diagrams.net**.
2. Prototype UI with **v0.dev**, **Lovable**, **Bolt.new**, **Dora**, **Replit**, or **10Web**.
3. Connect workflows using **n8n**, **Make**, **Zapier**, **Bardeen**, **Gumloop**, or **Lindy AI**.
4. Use **GitHub Copilot**, **Cursor**, **Windsurf**, **Claude Code**, or **Kilo Code** with repo gates and diffs.
5. Audit data movement with governance tools before anything touches real user data.

## 5. Weird Web Reset

1. Five minutes of **Radio Garden**, **Earth Nullschool**, or **WindowSwap**.
2. One odd interaction: **Pointer Pointer**, **Patatap**, **HackerTyper**, **The Useless Web**, or **Neal.fun**.
3. If you need a time capsule, send one small note through **FutureMe**.
4. Then return to the actual task. No feed doomscrolling.
